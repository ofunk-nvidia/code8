= Code8
:toc:
:sectnums:

Code8 is a VS Code extension prototype for an AI coding agent similar in spirit to Cline. It uses NVIDIA NGC / NIM as an OpenAI-compatible model provider and keeps secrets out of the repository.

== Current Scope

The first milestone is a compact, reviewable MVP:

* VS Code Activity Bar view with a chat composer.
* NVIDIA NGC / NIM chat-completions provider.
* Dynamic NVIDIA NIM model selection with provider discovery and official NVIDIA docs fallback.
* Agent loop with structured actions for listing files, reading files, writing files, replacing text, and proposing terminal commands.
* Persistent agent sessions with stop/reset controls.
* Human approval before workspace writes and terminal commands.
* AsciiDoc documentation for architecture, provider configuration, security, and Omnistation build flow.

== Quick Start

[source,powershell]
----
npm install
npm run build
----

Open this folder in VS Code and press `F5` to launch an Extension Development Host.

Set the API key through the command palette:

[source,text]
----
Code8: Set NVIDIA NGC API Key
----

The key is stored in VS Code SecretStorage. You can also set `NGC_API_KEY` in the extension host environment for local experimentation.

Select a hosted NIM model through:

[source,text]
----
Code8: Select NVIDIA NIM Model
----

The Code8 view also has `Model`, `Sync`, `Stop`, and `New` controls for changing models, refreshing the catalog, cancelling a run, and starting a fresh agent session.

== Configuration

[cols="1,2,2",options="header"]
|===
| Setting | Default | Purpose

| `code8.ngc.baseUrl`
| `https://integrate.api.nvidia.com/v1`
| OpenAI-compatible NVIDIA NIM base URL.

| `code8.ngc.model`
| `meta/llama-3.3-70b-instruct`
| Model ID sent to chat completions.

| `code8.ngc.modelCatalogUrl`
| `https://docs.api.nvidia.com/nim/reference/models-1`
| Official NVIDIA hosted NIM catalog page used when the provider model endpoint is unavailable.

| `code8.ngc.modelCatalogCacheMinutes`
| `60`
| Cache duration for the discovered NIM model list.

| `code8.agent.maxSteps`
| `8`
| Maximum model/tool iterations per request.

| `code8.agent.autoApproveRead`
| `true`
| Allows workspace file listing and reads without prompting.

| `code8.agent.requireApprovalForWrites`
| `true`
| Prompts before file creation or modification.

| `code8.agent.allowTerminalCommands`
| `false`
| Enables proposed terminal commands; each command still requires approval.
|===
