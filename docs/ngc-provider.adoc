= NVIDIA NGC Provider
:toc:
:sectnums:

== Provider Contract

Code8 talks to NVIDIA NGC / NIM through the OpenAI-compatible chat-completions API.

[source,text]
----
POST https://integrate.api.nvidia.com/v1/chat/completions
Authorization: Bearer <NGC_API_KEY>
Content-Type: application/json
Accept: text/event-stream
----

The default base URL is configurable through `code8.ngc.baseUrl`.

== Model

The default model is:

[source,text]
----
meta/llama-3.3-70b-instruct
----

Change it through `code8.ngc.model` when another NVIDIA-hosted NIM is preferred.

== Model Discovery

Code8 does not keep a fixed list of model IDs. It discovers models in this order:

1. `GET {code8.ngc.baseUrl}/models` using the configured NGC API key when available.
2. The official NVIDIA LLM API catalog page configured by `code8.ngc.modelCatalogUrl`.
3. The currently configured model ID if both discovery paths are unavailable.

The fallback page is used because NVIDIA documents the hosted LLM API models there and the catalog changes over time.

Use either the command palette or the Code8 view to select models:

[source,text]
----
Code8: Select NVIDIA NIM Model
Code8: Refresh NVIDIA NIM Model Catalog
----

== Secret Handling

Use the command palette command:

[source,text]
----
Code8: Set NVIDIA NGC API Key
----

The key is stored in VS Code SecretStorage. The repository intentionally does not contain `.env` files or API keys.
