= Omnistation Build Workflow
:toc:

== Requirement

The Cline-based Code8 branch expects a modern Node.js runtime. Use Node.js 20.20+ or Node.js 22.22+ for dependency install, type generation, webview build, and VSIX packaging.

If Omnistation is still on Node.js 18, upgrade or activate a newer Node runtime before building this branch.

== Build Commands

[source,bash]
----
scripts/omnistation-build.sh
----

The script installs an isolated Node.js runtime below `$HOME/workspace/tools`, verifies the Node.js tarball checksum, points `node-gyp` at the local Node headers, prepares local `grpc-tools` and `ripgrep` binary caches, installs root and `webview-ui` dependencies, and then packages the extension.

The default runtime is the latest Node.js 24.x release available from `nodejs.org`. Override it when needed:

[source,bash]
----
CODE8_NODE_VERSION=v24.11.1 scripts/omnistation-build.sh
CODE8_NODE_MAJOR=22 scripts/omnistation-build.sh
----

If Omnistation cannot reach `node-precompiled-binaries.grpc.io`, pre-seed this file:

[source,text]
----
$HOME/workspace/tools/cache/grpc-tools/v1.13.1/linux-x64.tar.gz
----

The build script points npm at `$HOME/workspace/tools/cache` via `npm_config_grpc_tools_binary_host_mirror`.

If Omnistation cannot reach GitHub release assets, pre-seed the ripgrep cache:

[source,text]
----
/tmp/vscode-ripgrep-cache-1.17.0/ripgrep-v15.0.0-x86_64-unknown-linux-musl.tar.gz
----

`npm run package:vsix` runs the Cline production package script and then creates:

[source,text]
----
code8-0.3.0.vsix
----

== Verification Checklist

* Provider selector contains `NVIDIA NGC / NIM`.
* Selecting the provider locks Base URL to `https://integrate.api.nvidia.com/v1`.
* Entering an NVIDIA API key enables model refresh.
* Refreshed models appear in a dropdown.
* Starting a task still uses the Cline approval/diff/terminal workflow.
